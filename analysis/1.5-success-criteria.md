### Levels and Layout

| Ref. No. | Criterion             | Justification                                                                                                                                                                        | Reference                                                                                                             |
| -------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------- |
|          | Split Screen          | The game will be online split screen, and so both players can see both players perspective.                                                                                          | <ul><li><a href="1.4a-features-of-the-proposed-solution.md#split-screen">Split Screen</a></li></ul>                   |
|          | Two Separate Worlds   | The players will not be playing the exact same game, as there will be slight differences in the minigames to prevent aspects of copying potentially present in a split screen game.  | <ul><li><a href="1.4a-features-of-the-proposed-solution.md#split-screen">Split Screen</a></li></ul>                   |
|          | Puzzles               | The game is a puzzle game and gives the player problems they need to solve to progress within the game.                                                                              | <ul><li><a href="1.4a-features-of-the-proposed-solution.md#puzzles-and-mini-games">Puzzles</a></li></ul>              |

|          | Equal Chance          | Although players will not be playing the exact same level, the route to the end for both players should be equal so neither player has an unfair advantage.                          | <ul><li><a href="1.4a-features-of-the-proposed-solution.md#split-screen">Split Screen</a></li></ul>                   |
